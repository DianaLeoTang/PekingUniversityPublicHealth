## **二、简答题解析（答题要点）**

76. **【环卫】简述环境卫生标准制定的原则。**
    - **科学性**：以环境流行病学、毒理学研究资料为基础。
    - **可行性**：考虑国家的经济、技术发展水平。
    - **安全性**：保障人群（包括敏感人群）健康和生态系统安全。
    - **统一性**：与相关标准协调统一。
    - **前瞻性**：适时修订，适应新的科学发现和社会需求。

77. **【职卫】简述职业病的特点。**
    - **病因明确**：由职业性有害因素引起。
    - **存在剂量-反应关系**：接触水平与发病风险相关。
    - **群发性**：在同一接触人群中常有一定发病率。
    - **可预防性**：控制或消除职业性有害因素可预防。
    - **疗效不佳**：多数职业病目前尚无特效治疗方法。
    - **立法性**：有法定的诊断标准和报告制度。

78. **【营养】简述断奶过渡期的定义和辅食添加的意义。**
    - **定义**：指婴儿由纯母乳喂养逐渐过渡到完全摄入母乳以外食物的时期，通常从4-6个月开始至2岁左右。
    - **意义**：
        1.  **补充营养**：弥补母乳中营养素的不足（如铁、维生素D）。
        2.  **发育需要**：训练婴儿咀嚼和吞咽能力，促进颌面部和牙齿发育。
        3.  **奠定基础**：为断奶和接受成人食物做准备，培养良好的饮食习惯。

79. **【流病】潜伏期在流行病学研究中有何用途?**
    1.  **确定暴露时间**：根据发病时间和潜伏期，推断患者可能的暴露时间，帮助追踪传染源。
    2.  **确定接触者的留验、检疫期限**：一般以该病的最长潜伏期为依据。
    3.  **确定预防接种时间**：如被狂犬咬伤后，应在潜伏期内接种疫苗。
    4.  **评价预防措施效果**：实施措施后，经过一个潜伏期，发病率下降，说明措施可能有效。
    5.  **判断流行强度**：短时间出现大量病人，且均在潜伏期内，提示为爆发。

80. **【流病】简述在筛检试验中，串联和并联对试验的灵敏度、特异度、阳性预测值、患病率、检出率的影响。**
    - **串联试验（依次进行，全部阳性才判为阳性）**：
        - **灵敏度**：降低（漏诊增加）。
        - **特异度**：升高（误诊减少）。
        - **阳性预测值**：升高。
    - **并联试验（同时进行，任一阳性即判为阳性）**：
        - **灵敏度**：升高（漏诊减少）。
        - **特异度**：降低（误诊增加）。
        - **阳性预测值**：降低。
    - **患病率**：是人群本身的特征，与试验方法无关。
    - **检出率**：并联试验的检出率高于串联试验。

## **三、论述题解析（答题要点）**

82. **【环卫】试述环境卫生学有哪些主要研究方法及不同研究方法间的区别联系。**
    - **主要方法**：
        1.  **环境流行病学方法**：在人群水平上研究环境因素与健康效应的关系。
        2.  **环境毒理学方法**：在实验室利用实验动物或体外试验研究环境因素的毒性、机制和剂量-反应关系。
        3.  **环境监测与评价方法**：检测和评价环境中污染物的水平、分布和变化趋势。
        4.  **健康危险度评价方法**：综合以上方法，定量评价环境有害因素对人群健康产生危害的概率及其程度。
    - **区别与联系**：
        - **区别**：
            - **层次不同**：毒理学（微观、机制）→ 流行病学（宏观、人群证据）→ 危险度评价（综合、预测与管理）。
            - **控制程度不同**：毒理学可严格控制条件；流行病学在真实世界观察，混杂多。
            - **目的不同**：毒理学揭示“可能性”与机制；流行病学验证“真实性”；监测提供基础数据；危险度评价服务于决策。
        - **联系**：
            - **相辅相成**：环境流行病学的结果需要环境毒理学提供机制解释；环境毒理学的发现需要环境流行病学在人群中验证。
            - **数据共享**：环境监测数据是流行病学研究和危险度评价的基础。
            - **目标一致**：共同服务于阐明环境与健康关系，为制定环境卫生标准、预防疾病提供科学依据。

83. **【职卫】影响粉尘对健康危害的主要因素以及粉尘所致的健康损害有哪些?**
    - **影响因素**：
        1.  **粉尘的化学组成**：决定其毒作用性质，如游离SiO₂含量越高，致纤维化能力越强。
        2.  **粉尘的浓度和暴露时间**：浓度越高、暴露时间越长，危害越大。
        3.  **粉尘的分散度**：分散度越高，在空气中悬浮时间越长，被吸入呼吸道的几率越大。
        4.  **粉尘的硬度、形状和溶解度**：影响其对呼吸道黏膜的机械损伤和清除速度。
        5.  **个体因素**：如健康状况、个人防护习惯等。
    - **健康损害**：
        1.  **呼吸系统疾病**：
            - **尘肺病**：最严重、最常见的职业病（如矽肺、煤工尘肺）。
            - **粉尘沉着症**。
            - **有机粉尘引起的肺部病变**（如棉尘病）。
            - **呼吸系统肿瘤**（如石棉肺患者肺癌和间皮瘤发病率增高）。
            - **粉尘性支气管炎、肺炎、哮喘等**。
        2.  **局部作用**：刺激皮肤、黏膜，引起皮炎、角膜炎、咽炎等。
        3.  **中毒作用**：含毒物的粉尘（如铅、砷）经呼吸道吸入可引起全身中毒。

84. **【营养】案例分析**
    **(1) 该次事件是否是食物中毒?如果是，是哪一类食物中毒，造成中毒的原因是什么?可疑食物和可疑餐次如何确定?**
    - **判断**：是食物中毒。
    - **类型**：**细菌性食物中毒**（根据剧烈呕吐、腹痛、腹泻等消化道症状，潜伏期短）。
    - **原因**：食用了被致病菌（特别是**金黄色葡萄球菌**，其肠毒素可引起剧烈呕吐）或其毒素污染的课间餐。
    - **确定可疑食物和餐次**：
        - **餐次**：所有病人均在10月12日上午食用过**课间餐**，而之前餐次无共同暴露，故可疑餐次为课间餐。
        - **食物**：通过**病例对照研究**或**回顾性队列研究**，比较发病组与未发病组进食各种食物的比例，计算OR值或RR值。若进食**豆浆和/或红豆糕**的OR/RR显著增高，则可确定其为可疑食物。
    **(2) 此类食物中毒流行病学特点是什么?病人的处理应注意哪些方面?**
    - **流行病学特点**：
        1.  潜伏期短，发病急剧。
        2.  所有病人有相似的临床症状（恶心、剧烈呕吐、腹痛、腹泻）。
        3.  发病与食用同一食物（课间餐）有关，未食者不发病。
        4.  停止供应可疑食物后，爆发即告终止。
        5.  人与人之间无直接传染。
    - **病人处理注意事项**：
        1.  **及时救治**：迅速送往医院，根据病情进行补液、纠正电解质紊乱及对症支持治疗。
        2.  **对症处理**：针对剧烈呕吐、腹泻等症状进行治疗。
        3.  **抗生素使用**：通常不需要使用抗生素，因为金黄色葡萄球菌食物中毒主要由肠毒素引起。
        4.  **隔离**：一般不需隔离。
        5.  **样本采集**：必要时采集病人呕吐物、粪便样本进行病原学检测。

85. **【流病】某研究人员为了研究吸烟对心肌梗塞发生的影响...**
    **(1) 研究设计类型**：**病例对照研究**。
    **(2) 资料分析**：
        - **计算各层的OR值**：
            - 脑力劳动者：OR = (92×16) / (40×12) = 1472 / 480 ≈ 3.07
            - 体力劳动者：OR = (48×10) / (6×20) = 480 / 120 = 4.00
        - **计算合并的OR值（未分层）**：
            - 合计：OR = (92+48)×(16+10) / (40+6)×(12+20) = (140×26) / (46×32) = 3640 / 1472 ≈ 2.47
        - **结果评价与解释**：
            - 分层后，脑力劳动者和体力劳动者的OR值（3.07和4.00）均高于合并OR（2.47），说明**体力活动**是一个**混杂因素**。
            - 未分层分析低估了吸烟与心肌梗塞的关联强度（存在负混杂）。
            - 应采用**分层分析**或**多因素Logistic回归**来控制体力活动的混杂作用，以得到吸烟与心肌梗塞关联的正确估计。
            - 结论：在控制了体力活动的影响后，吸烟是心肌梗塞的危险因素。

86. **【统计】两种药物联合治疗的药物临床试验中...**
    **(1) 资料特点**：
        - **协同作用**：两药联合使用的效应**大于**两药单独使用效应之和。
        - **拮抗作用**：两药联合使用的效应**小于**两药单独使用效应之和。
    **(2) 统计分析方法**：**两因素方差分析**（2×2析因设计方差分析）。通过分析**交互项**的统计学意义来判断是否存在协同或拮抗作用。
    **(3) 无效假设H₀**：**两种药物之间无交互作用**（即，药物A的效应不依赖于药物B的水平，反之亦然）。
