## **二、简答题答题要点**

76. **【环卫】简述环境卫生学的研究内容。**
    - 环境与健康关系的基础理论研究。
    - 环境因素（物理、化学、生物）对健康影响的规律研究。
    - 环境污染对健康的急、慢性危害及远期效应（致癌、致畸等）。
    - 环境卫生监督与监测的理论、技术和方法。
    - 环境卫生标准和政策的制定与评估。
    - 新技术、新方法在环境卫生领域的应用。

77. **【职卫】简述职业病诊断的依据。**
    - **职业史**：详细、可靠的职业病危害因素接触史。
    - **职业卫生现场调查资料**：工作场所职业病危害因素浓度（强度）、防护设施等。
    - **临床表现和实验室检查**：症状、体征及特异的实验室检查结果（如血铅、尿锰等）。
    - **排除其他类似疾病**：需与临床表现相似的非职业性疾病相鉴别。
    - **综合分析**：以上资料需符合国家颁布的《职业病分类和目录》及相应的诊断标准。

78. **【营养】简述营养素密度(INQ)的概念、意义以及优点。**
    - **概念**：INQ = (某营养素含量/该营养素参考摄入量) / (所产生的能量/能量参考摄入量)。
    - **意义**：评价食物营养价值的一个指标，表示单位能量食物所提供的某种营养素的丰富程度。INQ=1表示该食物该营养素与能量供给达到平衡；INQ>1表示该食物是该营养素的良好来源。
    - **优点**：
        1.  直观地比较不同食物在提供同等能量时，某种营养素的含量。
        2.  有助于指导人们选择营养素密度高的食物，改善膳食质量。
        3.  是评价食物营养价值的简明、实用指标。

79. **【流病】流行病学研究方法大致分为几类?各自在流行病学研究中有何用处。**
    - **描述性研究**：揭示疾病或健康状况的三间分布，提供病因线索，为卫生决策提供基础数据。
    - **分析性研究**：
        - **队列研究**：检验病因假设，计算发病率、RR、AR等，论证强度较高。
        - **病例对照研究**：探索疾病病因，特别适用于罕见病，可计算OR。
    - **实验性研究**：通过人为干预来验证病因假设或评价防治措施效果，是论证强度最高的方法。
    - **理论性研究**：通过数学模型模拟疾病流行过程，预测流行趋势，评价控制策略。

80. **【流病】简述与传统的病例对照研究相比，回顾性队列研究有何优缺点。**
    - **优点**：
        1.  **因果时序明确**：先有暴露，后有结局。
        2.  **可计算发病率**：能直接计算发病密度或累积发病率。
        3.  **信息偏倚较小**：暴露资料通常在结局发生前已记录，回忆偏倚小。
        4.  **可研究多种结局**：可同时研究暴露与多种疾病结局的关系。
    - **缺点**：
        1.  **不适用于罕见病**：若结局罕见，则需要非常大的样本和很长的随访期。
        2.  **实施难度大**：耗时、耗力、耗资。
        3.  **易存在失访偏倚**：随访时间长可能导致研究对象流失。

81. **【统计】请绘制一个图形，说明何为简单线性回归分析中的残差。概括性地反映残差大小的统计指标是什么?**
    - **图形说明**：在散点图上绘制出回归直线。对于任意一个数据点`i`，其观测值为`(X_i, Y_i)`，回归直线上的预测值为`(X_i, Ŷ_i)`。从点`(X_i, Y_i)`向回归直线作一条**垂直线**，这段垂直线的长度 `e_i = Y_i - Ŷ_i` 即为该点的**残差**。图形上表现为所有数据点到回归直线的垂直距离。
    - **反映残差大小的统计指标**：**残差的标准差** 或 **均方根误差（Root Mean Square Error, RMSE）**。该指标概括了所有观测点的观测值与回归直线预测值之间的平均离散程度。RMSE越小，说明回归直线对数据的拟合效果越好。

85. **【流病】某项研究...**
    **(1) 研究类型及依据**：**队列研究**。依据：研究开始时按暴露（是否习惯性运动）分组，然后随访观察结局（心肌梗塞）的发生情况。
    **(2) 计算与解释**：
        - **运动组发病率** `Ie = 350 / 10000 = 0.035`
        - **非运动组发病率** `I0 = 1500 / 20000 = 0.075`
        - **RR（相对危险度）** = `Ie / I0 = 0.035 / 0.075 ≈ 0.47`。含义：运动者发生心肌梗塞的危险是非运动者的0.47倍，即运动是保护因素。
        - **AR（归因危险度）** = `I0 - Ie = 0.075 - 0.035 = 0.04`。含义：在非运动者中，有0.04的发病率可归因于缺乏运动。
        - **AR%（归因危险度百分比）** = `(I0 - Ie) / I0 × 100% = 0.04 / 0.075 × 100% ≈ 53.3%`。含义：在非运动者中发生的MI，有53.3%可归因于缺乏运动。
        - **PAR（人群归因危险度）** = `It - Ie`。需先计算全人群发病率 `It`。全人群病例数=350+1500=1850，全人口数=10000+20000=30000，`It = 1850/30000 ≈ 0.0617`。`PAR = 0.0617 - 0.035 = 0.0267`。含义：在全人群中，有0.0267的发病率可归因于缺乏运动。
        - **PAR%（人群归因危险度百分比）** = `(It - Ie) / It × 100% = 0.0267 / 0.0617 × 100% ≈ 43.3%`。含义：在全人群发生的MI中，有43.3%可归因于缺乏运动。
    **(3) NNH（需改变人数）**：`NNH = 1 / AR = 1 / 0.04 = 25`。含义：每增加25名运动者，在观察期内可减少1例心肌梗塞病例。

86. **【统计】...**

    2.  **拟合Cox模型及解释**：
        ```r

        ```
        **HR及95%CI解释**：以“血管侵犯”为例，若其HR=3.00 (95%CI: 1.17-7.69)：

        - **图形**：通过`plot(ph_test)`查看Schoenfeld残差散点图与时间的拟合曲线，如果曲线大致平行于时间轴，则满足PH假设。
