# 中山大学353卫生综合考研趋势分析及高频考点总结

## 一、考试基本情况分析

### 1.1 题型及分值分布（2022年题型为例）

根据2022年真题考情分析，题型分布如下：

| 科目 | 题型与分值 |
|------|-----------|
| **统计学** | 选择题21题×2分=42分；简答题1题×10分；论述题1题×18分 |
| **流行病学** | 选择题21题×2分=42分；简答题2题×10分=20分；论述题1题×18分 |
| **营养与食品卫生学** | 选择题11题×2分=22分；简答题1题×10分；论述题1题×18分 |
| **职业卫生与职业医学** | 选择题11题×2分=22分；简答题1题×10分；论述题1题×18分 |
| **环境卫生学** | 选择题11题×2分=22分；简答题1题×10分；论述题1题×18分 |

**总分配置：**
- 选择题（总分150分）：环卫11题×2分=22分；职卫11题×2分=22分；营养11题×2分=22分；流病21题×2分=42分；统计21题×2分=42分
- 简答题（总分60分）：环卫1题×10分；职卫1题×10分；营养1题×10分；流病2题×10分=20分；统计1题×10分
- 论述题（总分90分）：环卫1题×18分；职卫1题×18分；营养1题×18分；流病1题×18分；统计1题×18分

### 1.2 历年题型变化趋势

**重要发现：**
1. **题型稳定性**：自2011年起，题型题量趋于稳定，题型保持为选择题、简答题、论述题三种
2. **分值比例**：选择题:简答题:论述题 = 150:60:90 = 5:2:3
3. **各科比重**：流行病学和统计学占比最大（各84分），其余三科各占60分

## 二、各科目高频考点总结

### 2.1 流行病学（84分，占比28%）

#### **超高频考点（几乎每年必考）**

1. **研究设计类型及应用**
   - 病例对照研究 vs 队列研究的特点、优缺点
   - 实验流行病学的设计原则
   - 横断面研究的应用场景

2. **效应指标计算与解释**
   - OR值、RR值的计算与意义
   - 归因危险度（AR）、归因危险度百分比（AR%）
   - 人群归因危险度（PAR）

3. **偏倚控制**
   - 选择偏倚、信息偏倚、混杂偏倚的识别
   - 随机化、盲法、匹配的作用
   - ITT分析（意向性分析）的目的

4. **疾病频率测量指标**
   - 发病率、患病率、死亡率、病死率的区别与应用
   - 发病密度的计算
   - 标化率（SMR）的应用

5. **筛检与诊断**
   - 灵敏度、特异度、阳性预测值、阴性预测值
   - ROC曲线分析
   - 筛检试验的评价

#### **高频考点**

6. **传染病流行病学**
   - 传染病三环节（传染源、传播途径、易感人群）
   - 潜伏期、传染期、隔离期的概念
   - 疫情调查方法

7. **病因推断**
   - Hill准则（病因推断标准）
   - 因果关联的判断
   - 生态学模型

8. **Logistic回归**
   - 适用条件和应用场景
   - 结果解释

#### **常考题型示例**

**选择题常考点：**
- 研究设计类型的选择
- 偏倚类型的判断
- 指标的适用场景

**简答题常考点：**
- 实验流行病学相对于观察性研究的优点
- 队列研究中嵌套病例对照研究的作用
- 随机化、盲法的作用

**论述题常考点：**
- 设计某疾病的流行病学研究方案
- 分析研究中可能存在的偏倚及控制方法
- 评价某筛检试验的效果

### 2.2 卫生统计学（84分，占比28%）

#### **超高频考点**

1. **假设检验**
   - t检验（单样本、配对、两独立样本）
   - 方差分析（单因素、多因素）
   - χ²检验（四格表、R×C表）
   - 非参数检验（秩和检验、符号秩和检验）
   - 检验方法的选择依据

2. **回归与相关**
   - 线性回归分析（简单、多重）
   - Logistic回归
   - 相关分析（直线相关、等级相关）
   - 回归系数的意义与解释
   - 量纲变换对回归系数的影响

3. **基本概念**
   - 总体与样本
   - 参数与统计量
   - 误差类型（系统误差、随机误差）
   - 变异指标（标准差、变异系数）

4. **研究设计**
   - 实验设计的基本原则（随机、对照、重复）
   - 样本量估计
   - 检验效能（Power）

#### **高频考点**

5. **统计描述**
   - 集中趋势指标（均数、中位数、众数）
   - 离散趋势指标（标准差、四分位数间距）
   - 正态分布及其应用

6. **抽样方法**
   - 随机抽样方法（简单、分层、系统、整群）
   - 抽样误差的控制

7. **统计图表**
   - 统计图的选择与绘制
   - 统计表的制作规范

#### **常考题型示例**

**选择题常考点：**
- 统计方法的选择
- 假设检验的原理
- 基本概念的理解

**简答题常考点：**
- 说明某统计方法的适用条件
- 解释统计量的意义
- 比较不同统计方法

**论述题常考点：**
- 设计统计分析方案
- 解释统计分析结果
- 评价统计学结论的合理性

### 2.3 营养与食品卫生学（60分，占比20%）

#### **超高频考点**

1. **营养素功能与缺乏症**
   - 蛋白质：必需氨基酸、蛋白质营养学质量评价（氨基酸评分、生物价）
   - 脂类：必需脂肪酸、脂肪酸分类、脂肪营养学质量评价
   - 碳水化合物：分类、功能、膳食纤维
   - 维生素：
     * 脂溶性维生素（A、D、E、K）的功能与缺乏症
     * 水溶性维生素（B族、C）的功能与缺乏症
     * **重点：叶酸缺乏与神经管畸形**
   - 矿物质：钙、铁、锌、碘、硒的功能与缺乏症

2. **铁代谢**
   - 储存铁（铁蛋白、含铁血黄素）vs 功能铁（血红蛋白、肌红蛋白）
   - 铁的吸收与影响因素
   - 缺铁性贫血

3. **膳食营养素参考摄入量（DRIs）**
   - 平均需要量（EAR）
   - 推荐摄入量（RNI）
   - 适宜摄入量（AI）
   - 可耐受最高摄入量（UL）
   - **效应评估法确定营养素需要量**

4. **食物中毒**
   - 细菌性食物中毒（沙门菌、副溶血性弧菌、金黄色葡萄球菌）
   - 化学性食物中毒（亚硝酸盐、有机磷农药）
   - 有毒动植物中毒（河豚、毒蕈）

#### **高频考点**

5. **能量代谢**
   - 基础代谢
   - 体力活动水平
   - 能量消耗计算

6. **特殊人群营养**
   - 孕妇、乳母营养需要
   - 婴幼儿营养
   - 老年人营养

7. **营养状况评价**
   - 膳食调查方法
   - 人体测量指标（BMI）
   - 生化检测指标

#### **常考题型示例**

**选择题：** 各类营养素的功能、缺乏症、食物来源
**简答题：** 营养素的生理功能、DRIs的概念
**论述题：** 特殊人群营养需求、营养缺乏病的防治

### 2.4 职业卫生与职业医学（60分，占比20%）

#### **超高频考点**

1. **生产性粉尘**
   - 粉尘的理化性质（分散度、游离二氧化硅含量、荷电性）
   - 粉尘对健康的影响（主要看分散度）
   - 矽肺的发病机制与预防

2. **职业中毒**
   - **铅中毒**：吸收途径、代谢、毒作用机制、生物标志物、防治
   - **苯中毒**：造血系统损害、慢性苯中毒诊断
   - **有机溶剂**：对神经系统的影响（**二硫化碳主要引起精神症状**）
   - 一氧化碳中毒机制
   - 有机磷农药中毒

3. **物理因素**
   - **噪声**：听力损伤、评价指标
   - 高温与中暑
   - **减压病**：氮气溢出导致

4. **职业病诊断与管理**
   - 职业病的诊断原则
   - 职业病分类（10类132种）
   - **我国职业卫生奠基人：邓铁涛**

5. **职业危害评价**
   - 职业卫生管理三要素
   - 职业危险度评价步骤
   - 双阈值化合物的剂量-反应曲线（U型或J型）

#### **高频考点**

6. **工作场所有害因素**
   - 化学性有害因素
   - 物理性有害因素
   - 生物性有害因素

7. **职业流行病学**
   - 职业性队列研究
   - 职业暴露评估

#### **常考题型示例**

**选择题：** 职业病的诊断、职业性有害因素、职业中毒
**简答题：** 粉尘危害性影响因素、职业病预防措施
**论述题：** 某职业病的发病机制及防治策略

### 2.5 环境卫生学（60分，占比20%）

#### **超高频考点**

1. **环境卫生学基本概念**
   - 环境卫生学的研究内容
   - 环境与健康的关系
   - 环境污染物的健康效应评价

2. **环境污染物**
   - **POPs（持久性有机污染物）**：斯德哥尔摩公约
   - 食物链/食物网中的能量和物质转移
   - 联合作用类型（相加、协同、拮抗）
   - **有机磷农药对胆碱酯酶的联合作用**

3. **毒理学基础**
   - **毒性指标**：LD50、LD100、LD0、LD01（**LD100毒性最大**）
   - 生物半衰期（**4-5个生物半衰期后趋于稳定**）
   - 剂量-反应关系
   - **代谢酶**：最早发现的机体与环境的代谢酶

4. **毒性试验**
   - 急性毒性试验
   - 慢性毒性试验
   - **致畸试验**：研究致畸作用
   - 致突变试验
   - 致癌试验

5. **健康风险评价**
   - **环境危险度评价的核心**
   - 危险度评价步骤：
     * 危害识别
     * 剂量-反应关系评定
     * 暴露评估
     * 风险特征描述
   - 已知健康效应探索有害效应的研究方法

#### **高频考点**

6. **水环境卫生**
   - 饮用水水质标准
   - 水污染与健康
   - 水的净化与消毒

7. **大气环境卫生**
   - 大气污染物及来源
   - 空气污染对健康的影响
   - 室内空气污染

8. **土壤卫生**
   - 土壤污染
   - 土壤自净

#### **常考题型示例**

**选择题：** 环境污染物、毒性指标、环境标准
**简答题：** 环境卫生学研究内容、污染物的健康效应
**论述题：** 环境健康风险评价、污染控制措施

## 三、考研趋势分析

### 3.1 总体趋势

1. **题型稳定**：近10年题型基本保持"选择+简答+论述"的三段式结构，分值比例稳定
2. **重应用轻记忆**：选择题虽多，但更侧重概念理解和应用，而非单纯记忆
3. **综合性增强**：论述题经常涉及多学科交叉，要求考生具备综合分析能力
4. **与实际结合**：题目越来越贴近公共卫生实践，如疫情调查、职业病防治、营养干预等

### 3.2 各科目出题特点

#### 流行病学
- 偏重方法学：研究设计、偏倚控制是永恒主题
- 计算题常见：OR、RR、归因危险度等指标计算
- 案例分析：给出研究背景，要求设计方案或评价研究

#### 统计学
- 基础扎实：概念题占比大，要求对基本原理有深刻理解
- 方法选择：给定研究目的和资料类型，选择合适统计方法
- 结果解释：给出统计分析结果，要求解读其临床/公共卫生意义

#### 营养学
- 营养素为核心：各类营养素的功能、缺乏症是重中之重
- 特殊人群：孕妇、儿童、老年人营养需求常考
- 实际应用：膳食指导、营养干预措施

#### 职业卫生
- 职业病三大类：粉尘、化学毒物、物理因素
- 重点毒物：铅、苯、有机溶剂
- 预防为主：强调三级预防，工程控制优先

#### 环境卫生
- 毒理学基础：LD50、剂量-反应关系、毒性试验
- 风险评价：健康风险评价的步骤和方法
- 污染控制：水、气、土壤污染及防治

### 3.3 备考建议

1. **抓大放小，重点突破**
   - 流病和统计各占28%，务必拿下基础分
   - 其余三科虽占比相同，但内容量差异大，合理分配时间

2. **理解为主，适度记忆**
   - 概念性知识要深刻理解，而非死记硬背
   - 在理解基础上，总结记忆框架

3. **真题为王，反复练习**
   - 真题重复率虽不高，但考点重复率极高
   - 至少做3遍真题，总结高频考点

4. **构建知识网络**
   - 各科目内部要形成知识体系
   - 注意学科间的交叉联系（如流病与统计、营养与环境）

5. **关注时事热点**
   - 公共卫生事件（如疫情）可能成为考题背景
   - 了解最新的公共卫生政策和进展
